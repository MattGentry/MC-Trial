Authentication

Authentication

No authentication is required to access the Glasses 3 API. The only requirement is to know the IP address of the Recording Unit.

Network

The Glasses 3 API can be accessed via a web browser on your computer, phone, or tablet. Connection to the specific Recording Unit over WLAN is required.

 

Access

HTTPS is not currently supported. It is not possible to make request calls from a secure to a non-secure connection.

Security

Additional security for the Glasses 3 API must be done on the network level. This can be done in two ways:

  • Secure the wi-fi connection to the Recording Unit with a password (TobiiGlasses)

  • Transfer the Recording Unit connection to a secure wi-fi connection.
